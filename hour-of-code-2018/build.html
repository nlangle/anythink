next, let's add a map variable to set the center coordinates and the zoom level:copy highlighted code

the latitude coordinates we are starting with are 39.9869599.

            </script>

the longitude coordinates are -104.8195388. 
            <script>

              var mymap = L.map('map').setView([39.9869599, -104.8195388], 18);

            </script>

These coordinates were retrieved from Google Maps for the Anythink Brighton location.

The zoom level for the map is set to 18. 
            <script>

              var mymap = L.map('map').setView([39.9869599, -104.8195388], 18);

            </script>

We can change the map center by supplying new lat/long coordinates. And, we can change the geographic area shown by adjusting the zoom level. a number less than 18 will zoom further out, showing a greater area.


next, we will add a JavaScript variable that sets the map tiles (the base layer):copy highlighted code 

            <script>

              var mymap = L.map('map').setView([39.9869599, -104.8195388], 18);

              
              var CartoDB_Voyager = L.tileLayer('https://{s}.basemaps.cartocdn.com/rastertiles/voyager/{z}/{x}/{y}{r}.png', {
               attribution: '© OpenStreetMap © CARTO',
               subdomains: 'abcd',
               maxZoom: 20
              }).addTo(mymap);

            </script>

at the end of this new line, there is a callback which adds our new variable to the map .addTo(mymap)

emojis in all the right places

now we are ready to add map markers! let's start with our favorite location. for this, we will use the yummy emoji .

first add the code to declare a variable that sets our yummy icon, then add the marker to the map:copy yummy icon code 

            <script>

              var mymap = L.map('map').setView([39.9869599, -104.8195388], 18);

              var CartoDB_Voyager = L.tileLayer('https://{s}.basemaps.cartocdn.com/rastertiles/voyager/{z}/{x}/{y}{r}.png', {
	               attribution: '© OpenStreetMap © CARTO',
                 subdomains: 'abcd',
                 maxZoom: 20
               }).addTo(mymap);

              
              var yummy = L.icon({
               iconUrl: 'images/yummy-emoji.png',
               iconSize: [50, 50],
              });

            </script>


then add the marker to the map:copy yummy marker code 

            <script>

              var mymap = L.map('map').setView([39.9869599, -104.8195388], 18);

            </script>


now, we need to set the location for your favorite location. use Google Maps to search for your place of choice.

once you find it on the map, you can extract the latitude/longitude coordinates from the URL in the web browser address bar. It looks like this for Anythink Brighton: 
            https://www.google.com/maps/place/Anythink+Brighton+Library/@39.986964,-104.8217328,17z/data=!3m1!4b1!4m5!3m4!1s0x876c6d39283c959b:0xef5fb06b58a64850!8m2!3d39.9869599!4d-104.8195388

look at the very end, see the !3d39.9869599!4d-104.8195388? Those are the coordinates.

ignore the !3d, and just copy your latitude number following it, all the way to the next !. like so:
(!3d39.9869599!4d-104.8195388)

Do the same for the longitude, ignore the !4d and copy it until the end. MAKE SURE TO INCLUDE THE DASH -
(!3d39.9869599!4d-104.8195388)

Use these numbers to set your yummyMarker variable
var yummyMarker = L.marker([latitude, longitude], {icon:yummy}).addTo(mymap);

last, make sure you set the text for the popup that appears when the marker is clicked. add the location name between the <b> </b> tags, and info about why you think it is so awesome after that:
yummyMarker.bindPopup('<b>name goes here</b><hr/>why you like it');
